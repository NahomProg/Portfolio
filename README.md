# Nahom Hailu - CS Student Portfolio

My personal portfolio website featuring a terminal mode, project showcase, and tech skills.

## Features

- Hidden terminal mode (press ` key)
- Fully responsive design
- Dark theme with gradient accents
- Project placeholders ready to fill
- Live GitHub API integration in terminal

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (ES6)
- GitHub REST API
